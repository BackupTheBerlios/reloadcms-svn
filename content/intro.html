ReloadCMS is a content management system (CMS) based on modules, Lang packs, libraries etc. Our features are:
* Smart and simple users management.
* Simple, secure and powerful administration CP.
* Full Multilanguage support.
* No Database usage (e.g. MySQL, PostgreSQL) - Based on files.
* Fully modular!!!

If you have just installed ReloadCMS please register first user, that will be automaticaly created as site administrator.
